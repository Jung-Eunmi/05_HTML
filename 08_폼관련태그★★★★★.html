폼 관련 태그

        form 태그는 html 에서 사용자가 값을 입력할 수 있는 양식을 제공하는 태그이다.
        form 태그의 input 태그를 사용해서 사용자가 입력한 데이터를 Server 로 전달하는 역할을 하게 된다.
        action 속성
            - 폼에 입력 된 값들을 전송받을 서버의 클래스명을 입력
        method 속성
            - get/post 방식으로 서버 전송 방식을 지정
            - get 방식 : url 에 넘어가는 데이터가 보인다.
            - post 방식 : url 에 넘어가는 데이터가 감춰진다.
